# 草皮用量计算器 — 完整视觉重构设计文档

## 设计哲学

**"克制即力量"** — 参照 Stripe、Apple、Notion 的设计语言：大面积留白、排版驱动视觉层级、颜色只做点睛、每个元素都有呼吸空间。

---

## 1. 色彩系统

```
背景层级：
  --bg-page:     #fafaf9    页面底色（暖灰，非纯白）
  --bg-card:     #ffffff    卡片/容器
  --bg-input:    #f5f5f4    输入框底色
  --bg-result:   #f0fdf4    结果区底色（极淡绿）

文字层级：
  --text-primary:   #1c1917  主文字（暖黑，非纯黑）
  --text-secondary: #78716c  次要文字
  --text-tertiary:  #a8a29e  辅助文字

品牌色（仅用于强调）：
  --green-500:  #22c55e    主强调（结果数字、活跃状态）
  --green-600:  #16a34a    悬停态
  --green-700:  #15803d    按钮背景

边框：
  --border:     #e7e5e4    默认边框（暖灰）
  --border-focus:#22c55e   聚焦边框
```

**核心原则**：页面 90% 是暖灰+白，绿色只出现在需要用户注意的地方（结果数字、活跃按钮、链接）。

---

## 2. 排版

```
字体栈：'Inter', system-ui, -apple-system, sans-serif

H1:  2rem / 700 / letter-spacing: -0.025em   页面标题
H2:  1.375rem / 600 / letter-spacing: -0.01em  区块标题
Body: 1rem / 400 / line-height: 1.6            正文
Small: 0.8125rem / 400                          辅助文字
Label: 0.75rem / 600 / uppercase / letter-spacing: 0.05em  表单标签
Result: 2.5rem / 800 / letter-spacing: -0.03em  结果数字
```

**核心原则**：标题用紧凑字距营造高级感，结果数字用超大字号+超紧字距制造视觉冲击。

## 5. 核心组件设计

### 5.1 输入框

```
┌──────────────────────────────┐
│  LENGTH                      │  ← 大写标签，0.75rem，600字重
│  ┌────────────────────┬────┐ │
│  │ 50                 │ ft │ │  ← 输入区白底，单位后缀浅灰底
│  └────────────────────┴────┘ │
└──────────────────────────────┘
```

- 输入框：白底，1.5px 暖灰边框，聚焦时绿色边框+绿色外发光
- 单位后缀：浅灰背景 `#f5f5f4`，左侧 1px 分隔线
- 输入字号 1rem，数字右对齐
- 标签与输入间距 6px

### 5.2 结果区

```
┌──────────────────────────────────┐
│  YOUR RESULTS                    │
│                                  │
│         1,500                    │  ← 2.5rem, 800字重, 绿色
│         sq ft total area         │  ← 0.8125rem, 灰色
│                                  │
│  ─────────────────────────────── │  ← 细分割线
│                                  │
│  Sod Rolls        165           │
│  Pallets           4            │  ← 左标签右数值
│  Cost Range    $495–$1,320      │
│                                  │
└──────────────────────────────────┘
```

- 背景：极淡绿 `#f0fdf4`，1px `#bbf7d0` 边框
- 主结果数字：超大、绿色、居中
- 详细数据：左右两栏布局，标签灰色，数值深色加粗

### 5.3 按钮组（Roll Size / Waste Factor）

```
  ┌──────────┬──────────┬──────────┐
  │ Standard │   Mini   │  Large   │
  └──────────┴──────────┴──────────┘
       ▲绿色背景白色字  ▲白底绿字
```

- 整体：1.5px 绿色边框圆角容器
- 活跃项：绿色背景 + 白色文字
- 非活跃项：白底 + 绿色文字
- 无单独边框分割，用背景色区分

### 5.4 折叠面板（Advanced Options）

```
  ▸ Advanced Options                    ← 绿色文字，箭头图标
  ────────────────────────────────────  ← 展开时显示分割线
  [展开的内容]                           ← 展开动画 200ms
```

### 5.5 手风琴（FAQ）

```
  ┌──────────────────────────────────┐
  │ How much sod do I need?      ▾  │  ← 白底，悬停浅灰
  ├──────────────────────────────────┤
  │ The amount depends on...         │  ← 展开内容，浅灰底
  └──────────────────────────────────┘
```

- 闭合项：白底，1px 暖灰边框
- 展开项：浅灰底 `#fafaf9`，左侧 3px 绿色指示条
- 图标：chevron，展开时旋转 180°

## 8. 动效

```
输入聚焦：border-color 150ms + box-shadow 150ms
按钮悬停：background-color 150ms
卡片悬停：box-shadow 200ms + translateY(-1px)
折叠展开：max-height 200ms ease
手风琴展开：max-height 250ms ease
页面加载：opacity 0→1, translateY(8px→0), 300ms ease
```

**原则**：所有动效 < 300ms，只用于功能反馈，不用于装饰。

---

## 9. 响应式断点

```
< 640px:  单栏，全宽卡片，底部导航
640-767px: 单栏，卡片有内边距
≥ 768px:  单栏居中 640px，顶部导航，隐藏底部导航
≥ 1024px: 单栏居中 640px，内容更宽松
```

**不做**：侧边栏、双栏布局、右侧栏。保持单栏的专注感。

---

## 10. 不做的事

- ❌ 深色/暗色模式（自然清新风格不需要）
- ❌ 大面积渐变背景
- ❌ 装饰性 SVG 图案
- ❌ 动画背景或粒子效果
- ❌ 多栏布局
- ❌ 侧边导航栏
- ❌ 右侧广告栏
- ❌ 超大 Hero 图片
